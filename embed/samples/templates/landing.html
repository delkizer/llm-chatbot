spo-chatbot Framework Samples

Web Component(<spo-chatbot>)가 다양한 프레임워크에서 동작함을 검증하는 샘플 모음

embed.js 빌드: cd embed && npm run build | 게이트웨이 실행: cd embed/samples && uvicorn gateway:app --port 8280 --reload
V
Vanilla HTML
프레임워크 없이 순수 HTML/JS만으로 Web Component를 사용하는 최소 구성 예제
정적 HTML | script 태그 직접 로드
V
Vue 3
Vue 3 Composition API 환경에서 isCustomElement 설정으로 Web Component 공존 검증
Vite 빌드 | compilerOptions.isCustomElement
R
React
React 환경에서 ref 기반 속성 바인딩으로 Web Component 동적 제어 검증
Vite 빌드 | useRef + setAttribute
S
Svelte
Svelte의 네이티브 커스텀 엘리먼트 지원으로 가장 자연스러운 통합 확인
Vite 빌드 | 네이티브 바인딩
N
Next.js
SSR 환경에서 dynamic import + ssr:false로 CSR 전용 로딩하여 hydration 문제 우회
정적 내보내기 (output: export) | dynamic import
A
Angular
Angular CUSTOM_ELEMENTS_SCHEMA로 커스텀 엘리먼트 인식 및 양방향 바인딩 검증
Angular CLI 빌드 | CUSTOM_ELEMENTS_SCHEMA
H
HTMX
HTMX와 Web Component의 공존 확인. Jinja2 서버 사이드 렌더링으로 토큰/설정 주입
Jinja2 SSR | HTMX 이벤트 비간섭 확인
I
iframe
iframe 격리 환경에서 postMessage 기반 호스트-챗봇 통신 패턴 검증
정적 HTML | postMessage 통신
spo-chatbot embed | Gateway: localhost:8280
